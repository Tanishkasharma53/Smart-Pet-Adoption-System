Project Overview

The Smart Pet Adoption System is a web application designed to simplify the pet adoption process. Users can browse pets, view profiles, and submit adoption requests. The system includes a smart compatibility algorithm that matches adopters with pets based on their preferences, living conditions, and lifestyle factors.


Features

Two user roles: Adopter and Admin

Browse and filter pet profiles with details like species, age, and activity level

Send adoption requests to the admin

Admin dashboard to view and manage adoption requests

Smart compatibility scoring to match adopters with suitable pets

Responsive UI with modals and dynamic content

Designed with user-friendly navigation and interactive elements


Technologies Used

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: Python

Database: SQLite

Other: API integration, UI/UX design, business logic, compatibility algorithm
